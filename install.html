Installation Guide

Warning

MacOS install warning

Currently due to a bug pride fonts will not work on MacOS.


Step 1. Download

First download the zip file with the built fonts from the homepage. Then extract the zip file


Step 2. Install on GNU/Linux

First install the noto-emoji package for you're distribution, for example this is the arch linux package.

Now copy the downloaded font file to /usr/share/fonts/noto/NotoColorEmoji.ttf, this will install the new emoji font!
